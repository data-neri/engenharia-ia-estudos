# Estudos de Engenharia de IA

Repositório dedicado ao estudo e implementação de arquiteturas avançadas de Inteligência Artificial, focado em RAG (Retrieval-Augmented Generation), orquestração de agentes e LangChain.

## 🚀 Tecnologias Utilizadas

- **LangChain & LangGraph:** Orquestração de fluxos complexos e agentes cíclicos.
- **OpenAI & HuggingFace:** Modelos de linguagem e embeddings.
- **FAISS:** Vector database para busca semântica eficiente.
- **Python:** Linguagem base para toda a engenharia de dados e IA.

## 📂 Estrutura do Projeto

- **Arquiteturas_RAG_com_LLMs:** Implementações fundamentais de RAG.
- **LangChain_Avançada:** Técnicas avançadas de processamento de documentos e memória.
- **LangChain_e_Python:** Integração de Python com o ecossistema LangChain.
- **LangGraph_Orquestrando_agentes_e_multiagentes:** Desenvolvimento de sistemas multi-agentes e grafos de decisão.

## 🛠️ Como Executar

1. Clone o repositório:
   ```bash
   git clone <url-do-seu-repositorio>
   ```
2. Crie um ambiente virtual e instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure suas chaves de API no arquivo `.env` (use o `.env.example` como base).
